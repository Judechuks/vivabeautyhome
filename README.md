packages
Installed packages for backend

1. npm init -y (to initialize the package.json file)
2. npm i express dotenv nodemon
3. npm i mongoose (ODM - object data modeling library for MongoDB and Nodejs)
4. npm i validator (popular js library used for data validation and sanitization in Node.js. It provides a collection of robust, pre-built validation and sanitization functions for common use cases, such as validating email addresses, URLs, and much more.)
5. npm i bcryptjs (library in Node.js for securely hashing and verifying passwords. It is widely used in authentication systems to protect user credentials. The hashing process ensures that even if a database is compromised, the plain-text passwords cannot be easily retrieved.)
6. npm i jsonwebtoken (used to securely transfer information between two parties (i.e., the client and the server) in a compact verifiable format. Its primary purpose is for authentication and authorization, and information exchange)
7. npm i cookie-parser (The cookie-parser middleware is a Node.js package used with Express.js applications to parse cookies attached to an incoming HTTP requests. It makes it easier to work with cookies by converting them into an easily accessible object (req.cookies) within the request object)
8. npm i crypto (to generate random numbers)
9. npm i nodemailer (for sending email)
